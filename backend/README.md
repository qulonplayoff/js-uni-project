# Лабораторна робота №2 - Бекенд без БД
**Виконав:** Шеремета Назар, група кб-11
**Варіант:** 15 (Інвентаризація носіїв інформації)

## Як запустити локально:
1. Відкрити термінал у папці `backend`.
2. Встановити залежності командою: `npm i`
3. Запустити сервер у режимі розробки: `npm run dev`
4. Сервер працюватиме за адресою: `http://localhost:3000`

## Реалізовані сутності:
1. **Users** (`/api/users`) - Користувачі (реалізовано через тришарову архітектуру).
2. **Devices** (`/api/devices`) - Пристрої (флешки, диски).

## Приклади тестування через curl (у PowerShell):
**Створення пристрою:**
`curl.exe -i -X POST http://localhost:3000/api/devices -H "Content-Type: application/json" -d '{"serial":"USB-999", "type":"Флешка", "owner":"Nazar", "status":"Вільно"}'`

**Отримання списку пристроїв:**
`curl.exe -i http://localhost:3000/api/devices`
